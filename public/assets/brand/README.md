# Brand Assets

Drop official Prowl Hub assets in this folder so the site can use them directly.

Recommended filenames:
- `logo-mark.svg` (used in nav)
- `logo-wordmark.svg` (optional for hero/footer)
- `mascot.png` (optional for hero art)
- `social-card.png` (optional for metadata previews)

The current app references `logo-mark.svg`.
